My one question:

👉 Why do you use <%= ... %> for project data instead of <%- ... %> here, and what’s the difference between the two in EJS?


we need the actual result without the html code so escaping it is useful

Exactly! 👏 That’s the key distinction in EJS:

<%= ... %> → Escapes HTML → outputs plain text safely (so any HTML tags in your data are treated as text, not executed)

<%- ... %> → Does not escape HTML → outputs raw HTML (so tags in your data are rendered as HTML in the page)

So in your case, <%= project.title %> and <%= project.organization %> are escaped because you want safe, plain text in the list, not any unintended HTML that could break your page or create a security risk.
